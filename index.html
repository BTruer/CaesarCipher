Caesar Cipher

How it works:

  Input plain text
  Cipher is applied:
    will shift every character by
    some value (this is called the key)
      if key=1 then A becomes B
      if key=2 then A becomes C
      if key=3 then A becomes D and so on
  Output cipher text
